# behavioural-analysis
🛍️ Customer Behavior Analysis

This project focuses on analyzing customer shopping behavior using a combination of Python (Pandas), SQL, and Power BI to extract insights and build an interactive dashboard.

📌 Project Overview

The goal of this project is to understand customer purchasing patterns, spending behavior, and preferences by performing:

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
SQL-based business queries
Interactive dashboard creation in Power BI
🧰 Tech Stack
Python (Pandas)
MySQL (for querying and data storage)
Power BI (for visualization and dashboarding)
